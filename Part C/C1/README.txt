Employee Management System Using JDBC

Files Included:
1. Employee.java
2. DBConnection.java
3. EmployeeManagement.java
4. database.sql

Compile:
javac -cp ".:mysql-connector-j-9.3.0.jar" *.java

Run:
java -cp ".:mysql-connector-j-9.3.0.jar" EmployeeManagement

Download MySQL JDBC Driver:
https://dev.mysql.com/downloads/connector/j/
